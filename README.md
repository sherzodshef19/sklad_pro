# Sklad.pro — Омбор бошқаруви тизими

PHP ва Bootstrap асосида ишлаб чиқилган складни бошқариш тизими.

---

## 📦 Лойиҳани ишга тушириш

1. `db.sql` файлини MySQL базасига импорт қилинг
2. `includes/db.php` файлида маълумотлар базаси созламаларини ўзгартиринг
3. Дефолт кириш: `admin` / `admin123`

---

## ✨ Имкониятлар

- 📦 Товарлар омборини бошқариш
- 📥 Кирим — омборга товар қабул қилиш
- 🛒 Кўп маҳсулотли сотув (корзинка)
- 🖨️ Термо-принтерга мос чек чиқариш
- 📜 Сотувлар тарихи (фильтр билан)
- 👥 Мижозларни бошқариш
- ⚙️ Созламалар ва маълумотлар базаси бэкапи

---

## ⚙️ Тизим талаблари

- PHP 7.2+
- MySQL 5.7+

---

## 📁 Файл структураси

```
sklad/
├── index.php             # Бош саҳифа (статистика)
├── products.php          # Товарлар
├── income.php            # Кирим
├── outcome.php           # Янги сотув (корзинка)
├── sales.php             # Сотувлар тарихи
├── customers.php         # Мижозлар
├── check.php             # Чек чиқариш
├── settings.php          # Созламалар
├── login.php             # Кириш
├── logout.php            # Чиқиш
├── db.sql                # Маълумотлар базаси схемаси
├── includes/
│   ├── db.php            # DB улашиш
│   ├── init.php          # Сессия ва аутентификация
│   ├── header.php        # Сайдбар + меню
│   └── footer.php        # Ёпиш тегларм
└── assets/
    ├── css/              # Bootstrap + style.css
    └── js/               # Bootstrap JS
```
