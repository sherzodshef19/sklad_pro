# Sklad.pro - Inventory & Sales Management System

A PHP-based warehouse management system built with Bootstrap.

## Features
- Product inventory management
- Income (stock receiving)
- Multi-product sales with cart system
- Printable receipts (thermal printer ready)
- Sales history with filters
- Customer management
- Settings & backup

## Requirements
- PHP 7.2+
- MySQL 5.7+

## Installation
1. Import `db.sql` into your MySQL database
2. Configure DB credentials in `includes/db.php`
3. Default login: `admin` / `admin123`
